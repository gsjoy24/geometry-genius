1. What is DOM and what is the purpose of DOM?

Ans: The DOM stands for Document Object Model. It represents the interface of a HTML or XML file. Using DOM we can access every element in the document and also can manipulate that. Such as delete, create or style elements.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: There are 5 ways to select HTML elements a DOM using selectors.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Ans: Event bubbling is a method or concept on JavaScript DOM. When a button is clicked or triggered, the event also effects the parents of that button or the element by bubbles up.

4. What is a callback function and why will you use it?

Ans: Callback function are usually passed as an argument in another function. It called or executed after the parent function completely executed.

We need callback functions to make sure that a function is only going to get executed after one or more specific tasks are completed and not before its completion.
